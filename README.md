# PhoenixBOT
> Discord BOT with everything a FiveM Store needs.

## Installation

Windows:

```sh
Download NodeJs here: https://nodejs.org/en/download/
npm install discord.js --save
npm i mysql

node index.js
```
## Configuration

```sh
Edit config.js with your own settings

```
